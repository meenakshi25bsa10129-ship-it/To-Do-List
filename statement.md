Problem Statement Users often choose weak passwords. This tool helps evaluate and guide users to create stronger passwords by rating their current password and suggesting new modification.

Scope of the project CLI-based password strength evaluator with actionable suggestions.

Target users The main users here are developers or security engineers who want a small, clear function they can plug into their apps without hassle to handle password rules.

High-Level Features 1.Password validation 2.Strength scoring 3. Improvement suggestions
